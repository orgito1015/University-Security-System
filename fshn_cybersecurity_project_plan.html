Faza 3 — Funksionalitetet kryesore
3
Çfarë bën sistemi konkretisht
Features të sistemit të ri
Autentikimi 2FA
Login + kod SMS/email. Bllokiim pas 3 tentativave të dështuara. Session timeout automatik.
Monitor aktivitetit
Çdo login regjistrohet: IP, ora, pajisja. Detektim i hyrjeve nga vende/IP të pazakonta.
Sistemi i alerteve
Admin merr njoftime për aktivitet të dyshimtë. Raporti javor i sigurisë automatik.
Auditimi i të dhënave
Kush ka ndryshuar çfarë dhe kur. Log i pandryshueshem për notat dhe oraret.
